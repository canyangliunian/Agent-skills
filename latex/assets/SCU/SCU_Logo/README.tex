	1.	大家都会\LaTeX{}，好多学校都有自己的beamer主题, 但是overleaf上没有川大的模板
	
	2.  本模板基于overleaf上清华的模板进行调整，同时参考了南方科大的beamer模板
	
	3.	改动之处:基础色设置为川大红、校徽元素的添加、部分公式的改动、图片的替换、lstlisting相关内容的删除、参考文献格式的改动。所有的参考链接已注明在code中
	
	4.	后缀名为sty的文件是设置背景的关键，初学者尽量不要修改sty文件
	
	5.  本代码插入ctex宏包，所以适用于中文报告，英文报告删除ctex宏包即可
	
	6.	这个代码有四个关于字体设置的warning，英文环境下编译则不存在warning
	
	7.	感谢陈闯老师花费时间对这份模板的修订与建议，同时感谢马强老师对排版的建议
	
	8.	作为一个初学者，个人水平有限，其中的错漏之处还请海涵。希望这份beamer theme能够给大家提供一个参考，欢迎各位共同努力完善
